# 决策：MVP 功能范围

> **状态：已对齐当前实现**
> **对齐日期：2026-05-29**
> **当前原则**：以可运行核心闭环为准，未实现功能进入 backlog。

## 决策结果

当前 MVP 聚焦“本地浏览器单屏 AI 跑团核心体验”：

```
预设调查员 -> AI 叙事 -> 玩家行动 -> D100 检定 -> 状态更新 -> 存档恢复
```

## 当前 MVP 已实现

### 前端架构
- React 18 + TypeScript + Vite。
- 横屏游戏主界面。
- 低频功能收纳到菜单/抽屉。

### 角色系统
- 4 个预设调查员。
- 1-4 名调查员选择。
- 5 个职业定义、23 个技能定义。
- COC 风格属性、HP/MP/SAN、技能总值。

### AI DM 系统
- OpenAI / Anthropic / MiMo / 自定义 Provider。
- 系统提示词动态注入剧本、状态、角色卡。
- JSON 契约校验、格式修复重试、状态归一化。
- 推荐行动建议。

### D100 检定
- AI 请求检定。
- 前端计算阈值并掷骰。
- 检定结果回传 AI。

### 存档系统
- 保存当前游戏。
- 标题页/菜单读取最近有效存档。
- 存档管理弹窗列出有效存档。
- 支持指定载入和删除单个本地存档。
- `trpg-saves-v2` 新版存档。

### 质量护栏
- Playwright smoke tests。
- 覆盖标题页 -> 选角 -> 主界面、无 API Key 防崩溃、保存 -> 首页继续、非法存档过滤、D100 大失败优先规则。

### 剧本内容
- 「雾中消逝」5 个场景。
- 6 个 NPC 条目。
- 8 个线索物品。
- 场景/NPC/线索资产接入。

## 当前不在 MVP

- 自定义角色创建。
- 存档导入/导出。
- 完整战斗系统。
- SAN 疯狂系统。
- 背景音乐/音效系统。
- 多剧本切换。
- 多客户端同步。
- 后端服务器和账号系统。

## Backlog 优先级

| 优先级 | 项目 |
|------|------|
| P1 | 自定义角色创建 |
| P1 | 提示词外部化到 `/prompts` |
| P2 | 战斗/SAN 自动化 |
| P2 | 音效/BGM |
| P3 | 多人同步/后端代理 |
