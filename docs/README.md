# 策划文档目录

本目录存放所有策划相关文档，使用 Markdown 格式编写。**唐龙翔、刘晓可直接提交到此目录。**

## 文档规范

- 文件名使用中文或英文均可，避免空格（用 `_` 代替）
- 每个文档顶部注明作者、创建日期、最后更新日期
- 重大修改请写清楚改了什么（commit message 说明）

## 目录规划

```
docs/
├── 世界观简介.md        # 世界观 / 主题 / 故事背景
├── 玩法一页纸.md        # 核心玩法循环说明
├── 角色系统.md          # 属性 / 职业 / 背景设计
├── 剧本/
│   ├── ch1_起点.md     # 第一章剧本
│   └── ...
└── 风格参考/            # 美术风格参考图（茉莉整理）
```

> 以上为规划目录，按需创建，不必强求一次性建齐。
