# 文档目录

本目录存放当前项目的产品、技术和游戏设计文档，使用 Markdown 格式编写。当前文档已经与 Vite + React + TypeScript 代码结构对齐。

## 文档规范

- 文件名使用中文或英文均可，避免空格（用 `_` 代替）
- 每个文档顶部注明作者、创建日期、最后更新日期
- 重大修改请写清楚改了什么（commit message 说明）

## 当前文档

| 文件 | 作用 |
| --- | --- |
| `PRD.md` | 当前 MVP 产品范围、用户流程、验收标准 |
| `SPEC.md` | 当前技术架构、状态、AI、存档、骰子契约 |
| `GDD.md` | 《雾中消逝》游戏设计当前版本 |
| `GDD.html` | `GDD.md` 的轻量网页镜像 |
| `git使用指南.md` | Git 协作说明 |

## 后续规划

新增世界观、剧本、风格参考等资料时，优先保持与 `PRD.md`、`SPEC.md` 和 `TabletopRPG/` wiki 的事实一致。
